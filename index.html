My Projects

Heart Disease Prediction

This machine learning project predicts the likelihood of a person having heart disease based on medical parameters such as age, sex, blood pressure, cholesterol levels, chest pain type, and more.

I used the Scikit-learn library to build and evaluate multiple models including Logistic Regression, Decision Trees, and Random Forest. After comparing performance metrics like accuracy, precision, and recall, the Random Forest model achieved the best results.

The model was trained on the UCI Heart Disease dataset and includes essential steps like data preprocessing, feature scaling, train-test splitting, and model tuning.

Technologies Used: Python, Scikit-learn, Pandas, NumPy, Matplotlib

Resume Screening

This machine learning project automates the process of shortlisting resumes based on required skills and job roles. It classifies resumes into relevant job categories such as Data Scientist, Web Developer, or Software Engineer.

I used Natural Language Processing (NLP) techniques to clean and vectorize resume text using methods like tokenization, TF-IDF, and stopword removal. The processed text data was then fed into machine learning models such as Naive Bayes and Support Vector Machines (SVM) using Scikit-learn.

The model helps streamline the recruitment process by reducing manual effort and improving the consistency of resume evaluations.

Technologies Used: Python, Scikit-learn, Pandas, NLTK, TF-IDF
